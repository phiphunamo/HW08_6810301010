📋 รายละเอียดการบ้าน HW 8 (Personal Homepage)

📍 วัตถุประสงค์และการพัฒนา

เป็นการพัฒนาเว็บไซต์ประวัติส่วนตัว (Personal Website) จำนวนครบถ้วนตามเงื่อนไข 8 หน้า โดยเน้นการใช้เทคโนโลยีเว็บพื้นฐานอย่าง HTML5 และ CSS3 เพื่อออกแบบโครงสร้างหน้าเว็บที่รองรับการใช้งานจริง มีการจัด Layout เมนูด้วย Flexbox และจัดระเบียบโครงสร้างเนื้อหาข้อมูลส่วนตัว ประวัติการศึกษา และแบบฟอร์มการติดต่อ

🛠️ เทคโนโลยีและเครื่องมือที่ใช้งาน

  • HTML5 / CSS3: เขียนโครงสร้างและแต่งหน้าเว็บให้สวยงามด้วยโทนสีม่วง
  • Visual Studio: โปรแกรมหลักในการเขียนโค้ดและจัดการไฟล์ทั้งหมดใน Local Directory
  • Internet Information Services (IIS): ทำการติดตั้งและเปิดใช้งานบริการ Web Server บนเครื่อง Localhost (พอร์ต 8080) เพื่อจำลองการให้บริการเว็บไซต์จริง
